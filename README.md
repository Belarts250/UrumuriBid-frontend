# UrumuriBid - Modern Auction Platform

UrumuriBid is a transparent, secure, and user-friendly digital auction platform. It modernizes traditional bidding (Cyamunara) into a seamless online experience, providing a fair and accessible marketplace for all.

## 🚀 Features

- **Dynamic Landing Page**: Showcasing our professional solutions, experience timeline, and contact options.
- **Interactive Auctions Dashboard**: 
  - Real-time bid overview and metrics.
  - Consistent sidebar navigation across all user pages.
  - Integrated search and user notification system.
  - Live Google Maps integration for store locations.
- **My Deals & Auction Details**: Browse available auctions with ease and view detailed product information.
- **User Settings**: Manage personal profiles and account preferences.
- **Auth Experience**: Professional login and signup flows with test credentials support.

## 🛠️ Tech Stack

- **Frontend**: React with TypeScript
- **Routing**: React Router DOM v6
- **Styling**: Vanilla CSS (Custom premium design system)
- **Icons**: Lucide React
- **Build Tool**: Vite

## 📦 Getting Started

### Prerequisites

- Node.js (v16+)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Belarts250/Ecyamunara-frontend.git
   cd Ecyamunara-frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
